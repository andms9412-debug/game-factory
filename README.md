# 🎮 Game Factory 小遊戲工廠

持續產出小遊戲，目標上架 itch.io / Steam / Google Play。

## 專案結構

```
games/          — 每款遊戲一個資料夾
backlog/        — 遊戲企劃 spec
game-queue.md   — 開發排程
games-list.json — 遊戲清單（機器可讀）
```

## 技術棧

純 HTML + CSS + JS，無框架，手機優先，離線可玩。

詳見 [game-design-rules.md](game-design-rules.md)。
