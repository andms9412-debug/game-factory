# 猜台灣小吃

## 基本資訊

- **類型**：文字猜謎（Wordle 變體）
- **狀態**：測試中
- **建立日期**：2026-02-27

## 玩法

- 每日一題，猜一個三字的台灣小吃名稱
- 最多 6 次機會
- 每次猜測後顯示顏色提示：
  - 🟩 綠色：字正確且位置正確
  - 🟨 黃色：字正確但位置錯誤
  - ⬛ 灰色：字不在答案中
- 支援注音輸入與直接打字

## 目標受眾

- 台灣玩家、對台灣小吃有興趣的中文使用者

## 技術細節

- 單一 index.html（HTML + CSS + JS 內嵌）
- 深色主題、RWD 手機優先
- localStorage 存遊戲紀錄與統計
- 分享功能（複製 emoji 格子到剪貼簿）
