# 遊戲設計規範

所有 Game Factory 出品的遊戲必須遵守以下規範。

## 技術規範

- **純 HTML + CSS + JS**，不使用任何框架或建構工具
- 每個遊戲一個資料夾，至少包含 `index.html`
- **RWD 手機優先設計**，桌面也要能玩
- 使用 `localStorage` 儲存遊戲進度與紀錄
- **可離線運行**，不依賴外部 CDN 或第三方服務
- **itch.io 相容**：純靜態檔案，zip 上傳即可運行

## 設計規範

- **深色主題為預設**（背景 `#121213` 或類似暗色）
- 遊戲載入時間 < 1 秒
- 要有**分享功能**（複製結果到剪貼簿，類似 Wordle 分享格式）
- 遊戲結束後顯示統計資訊

## 無障礙要求

- 語意化 HTML（正確使用 `<button>`、`<header>`、`<main>` 等）
- 文字與背景對比度至少 4.5:1（WCAG AA）
- 可用鍵盤操作核心功能
- 重要狀態變化要有文字提示（不只靠顏色）

## 檔案結構範例

```
games/my-game/
├── index.html    （必要）
├── style.css     （可選，也可內嵌）
├── game.js       （可選，也可內嵌）
└── assets/       （可選，圖片音效等）
```
